# 🎀 MammoAI: Breast Cancer Detection System

**MammoAI** is a desktop application that leverages Deep Learning to assist in the early detection of breast cancer from mammogram images.

## ✨ Features
* **AI Analysis:** Uses a fine-tuned VGG16 model for high accuracy.
* **Modern UI:** Built with CustomTkinter for a sleek, dark-mode experience.
* **Fast & Portable:** Available as a standalone Windows installer.
